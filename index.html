Plugin FakeBox jQuery

Présentation du plugin

Ce plugin permet de remplacer un élément 
<select>
par une structure HTML permettant de manipuler bien plus facilement cet élément HTML.
Le css vous permet également de modifier selon vos contraintes tous les paramètres d'affichage du résultat.
Vous pouvez ainsi transformer une simple comboBox en drop-down avec une touche de fun en plus ! A travers les exemples ci-dessous, vous pourrez apprécier la facilité d'adaptation et de prise en main de ce plugin.

Exemple 1 : Default

							<script type="text/javascript">
								$('#wrapper0').fakebox();
							</script>
							
							<select id="wrapper0">
								<option value="0">item0</option>
								<option value="1">item1</option>
								<option value="2">item2</option>
								<option value="3">item3</option>
							</select>

Exemple 2 : Target Item 

							<script type="text/javascript">
								$('#wrapper1').fakebox({
									 targetEffect:"item",
									 openStyle : "inverse"
								});
							</script>
							
							<select id="wrapper1">
								<option value="">Sélectionner</option>
								<option value="0">item0</option>
								<option value="1">item1</option>
								<option value="2">item2</option>
								<option value="3">item3</option>
							</select>

Exemple 3 : Invert mode and easing
